算法 1 全局动量压缩（GMC）
输入：稀疏化压缩器 C (・)、工作节点数 K、迭代次数 T、初始模型参数 w₀、学习率 η、动量系数 β ∈ [0,1)、各工作节点训练数据集 Dₖ(∀k ∈ [K])初始化：w₋₁ = w₀，误差残差 e₀,ₖ = 0 (∀k ∈ [K])
for 迭代轮次 t ∈ [T] do工作节点侧（所有节点并行执行）：for 工作节点 k ∈ [K] parallelly do
从 Dₖ中随机采样批量为 b 的训练数据 Iₜ,ₖ，计算随机梯度：
∇f (wₜ; Iₜ,ₖ) = (1/b) * Σ(ξ∈Iₜ,ₖ) ∇f (wₜ; ξ)
计算中间误差项：
eₜ₊₁/₂,ₖ = eₜ,ₖ + ∇f (wₜ; Iₜ,ₖ) - (β/η) * (wₜ - wₜ₋₁)
生成稀疏向量 C (eₜ₊₁/₂,ₖ) 并发送至服务端
更新本地误差残差：
eₜ₊₁,ₖ = eₜ₊₁/₂,ₖ - C (eₜ₊₁/₂,ₖ)
从服务端接收参数更新量 wₜ₊₁ - wₜ
计算本地新参数：
wₜ₊₁ = wₜ + (wₜ₊₁ - wₜ)
end for
服务端侧：
接收所有工作节点发送的稀疏向量 C (eₜ₊₁/₂,ₖ)
更新全局模型参数：
wₜ₊₁ = wₜ - η * (1/K) * Σ(k∈[K]) C (eₜ₊₁/₂,ₖ)
将参数更新量 wₜ₊₁ - wₜ发送至所有工作节点
end for
输出：训练完成的模型参数 w_T
算法 2 增强型全局动量压缩（GMC+）
输入：稀疏化压缩器 C (・)、工作节点数 K、迭代次数 T、初始模型参数 w₀、学习率 η、分离系数 λ ∈ [0,1)、动量系数 β ∈ [0,1)、各工作节点训练数据集 Dₖ(∀k ∈ [K])初始化：w₋₁ = w₀，本地参数 w₀,ₖ = w₀，误差残差 e₀,ₖ = 0 (∀k ∈ [K])
for 迭代轮次 t ∈ [T] do工作节点侧（所有节点并行执行）：for 工作节点 k ∈ [K] parallelly do
从 Dₖ中随机采样批量为 b 的训练数据 Iₜ,ₖ，计算随机梯度（基于本地参数 wₜ,ₖ）：
∇f (wₜ,ₖ; Iₜ,ₖ) = (1/b) * Σ(ξ∈Iₜ,ₖ) ∇f (wₜ,ₖ; ξ)
计算中间误差项：
eₜ₊₁/₂,ₖ = eₜ,ₖ + ∇f (wₜ,ₖ; Iₜ,ₖ) - (β/η) * (wₜ - wₜ₋₁)
生成稀疏向量 C (eₜ₊₁/₂,ₖ) 并发送至服务端
更新本地误差残差：
eₜ₊₁,ₖ = eₜ₊₁/₂,ₖ - C (eₜ₊₁/₂,ₖ)
从服务端接收参数更新量 wₜ₊₁ - wₜ
计算本地新全局参数：
wₜ₊₁ = wₜ + (wₜ₊₁ - wₜ)
更新本地分离参数（分离操作）：
wₜ₊₁,ₖ = wₜ₊₁ - ληeₜ₊₁,ₖ
end for
服务端侧：
接收所有工作节点发送的稀疏向量 C (eₜ₊₁/₂,ₖ)
更新全局模型参数：
wₜ₊₁ = wₜ - η * (1/K) * Σ(k∈[K]) C (eₜ₊₁/₂,ₖ)
将参数更新量 wₜ₊₁ - wₜ发送至所有工作节点
end for
输出：训练完成的模型参数 w_T
